# 잘샀구매 서비스 API

## HOW to RUN CODE

```
conda create -n ML_API python=3.6.4

conda activate ML_API

cd ML
pip install -r requirements.txt
git clone https://github.com/ssut/py-hanspell.git

cd py-hanspell

python setup.py install

cd ..

```


## API 실행하는 명령어
python ML_API.py
